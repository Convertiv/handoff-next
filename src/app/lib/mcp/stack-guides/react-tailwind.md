# Stack guide: react-tailwind

- **Templates:** React TSX (`.tsx`) with typed props where applicable
- **CSS:** Tailwind utility classes; bind design tokens via CSS variables or `tailwind.config`
- **Config:** `defineReactComponent` or equivalent with `component` entry
- **Properties:** Align with component prop types / CSF args
- **Previews:** Story-style or live preview args in handoff config

Do not use Handlebars or Bootstrap grid classes unless migrating legacy blocks.
